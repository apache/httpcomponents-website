HttpComponents Downloads
================================

We recommend you use a mirror to download our release builds, but you <strong>
must</strong> verify the integrity of the downloaded files using signatures downloaded
from our main distribution directories. Recent releases (48 hours) may not yet be
available from the mirrors.

The `KEYS` link links to the code signing keys used to sign the product. The `pgp` link
downloads the OpenPGP compatible signature from our main site. The `sha512` link downloads
the SHA 512 checksum from the main site.
